## How to run

### 1. Install the dependencies

```
yarn install
```

### 2. Insert your API keys/secrets

Create a new file `.env.local` and fill in the following values

```
OPENAI_API_KEY=

PINECONE_API_KEY=
PINECONE_ENVIRONMENT=
PINECONE_INDEX_URL=
PINECONE_INDEX_NAME=
```

### 3. Run the project

```
yarn run start
```
